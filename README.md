# SecAI Journal

A Jekyll blog built with the [Chirpy](https://github.com/cotes2357/jekyll-theme-chirpy) theme, deployed via GitHub Pages at [secai-journal.github.io](https://secai-journal.github.io).

## Local Development

### Prerequisites

Install Ruby via rbenv (recommended on macOS):

```bash
brew install rbenv ruby-build
rbenv install 3.3.0
rbenv global 3.3.0
gem install bundler
```

### Setup

```bash
bundle install
```

### Running the site

```bash
bundle exec jekyll serve
```

Visit `http://127.0.0.1:4000` in your browser.

| Flag | Description |
|---|---|
| `--livereload` | Auto-refreshes the browser on file changes |
| `--drafts` | Also renders posts in `_drafts/` |

```bash
bundle exec jekyll serve --drafts
```

### Drafts

Create posts in `_drafts/` without a date prefix to preview them locally without publishing:

```bash
bundle exec jekyll serve --drafts
```

Move to `_posts/YYYY-MM-DD-title.md` when ready to publish.

## Writing Posts

Create files in `_posts/` following the `YYYY-MM-DD-title.md` naming convention:

```yaml
---
title: "My CTF Writeup: HackTheBox - SomeMachine"
date: 2026-03-25 10:00:00 +0100
categories: [CTF, HackTheBox]
tags: [privesc, linux, web]
---
```

## Site Structure

| Folder | Purpose |
|---|---|
| `_posts/` | Published blog posts (`YYYY-MM-DD-title.md`) |
| `_drafts/` | Work-in-progress posts — never published, local preview only |
| `_tabs/` | Sidebar navigation items (Categories, Tags, Archives, About) |
| `assets/img/` | Images — drop `avatar.png` here for the sidebar avatar |

### `_tabs/`

Chirpy-specific folder that defines the **sidebar navigation links**. Each file maps to one nav item via its frontmatter:

```yaml
---
icon: fas fa-info-circle   # Font Awesome icon
order: 4                   # Position in the sidebar
---
```

The only file you'll regularly edit here is `_tabs/about.md`. The others (`categories`, `tags`, `archives`) are auto-generated by Chirpy and rarely need changes.

## Deployment

Pushes to `main` automatically build and deploy via GitHub Actions. Monitor runs at:
`https://github.com/SecAI-Journal/secai-journal.github.io/actions`
